# 🚀 v1.0.0: The Dialer MVP

**⚠️ IMPORTANT NOTE:** This release marks the very first Minimum Viable Product (MVP) of DialerJazz. While it showcases the core design vision and foundational architecture, **it explicitly contains a lot of bugs** and many unpolished edge cases. Expect things to break as we continue to iterate!

## ✨ Highlights
- **Tinder for Sales UX:** The core swiping mechanism designed to gamify and accelerate outbound calling workflows.
- **Premium Aesthetics:** Implementation of the "Midnight Emerald" UI aesthetic, giving the dialer a sleek, modern, and high-end feel.
- **Open Source Foundation:** Fully set up with an MIT license, GitHub issue/PR templates, contributing guidelines, and a clean codebase to welcome community involvement.

## 🛠 Features
- **Frontend:** Fully responsive React application handling the swiping logic and UI states.
- **Backend:** Node.js Express server to orchestrate calling infrastructure.
- **BaaS Ready:** First-class support for InsForge, handling authentication, database schema, and real-time WebSocket syncing for call logs.
